How to run MinCAL

Step 1: Creating the configuration file

You must copy the contents of the file Conf/proj to the directory /etc/system.conf.d/

Step 2: Building the application

Change directory to /pr/Src and run make.

Step 3: Running the service

Run the command service run /pr/Src/proj -args init, as root, in order to initialize the service.
